# Keen Keeper

Keen Keeper is a friend relationship tracker built with Next.js.

## Description

Keep track of your friends and your interactions with them. Keen Keeper is a Next.js app that helps you stay on top of your friendships — set how often you want to check in with each friend, log your calls/texts/video chats, and see who you're overdue to reach out to before too much time passes.

## Built with

- Next.js (App Router)
- React Context API for state
- Tailwind CSS + DaisyUI
- react-icons
- react-toastify (for notifications)
- react-spinners (loading spinner)
- localStorage (so your data doesn't disappear on refresh)
- Deployed on Vercel

## Key features

- **Dashboard** — see all your friends, their status, and quick stats (total friends, who's on track, who needs attention)
- **Friend detail pages** — each friend has their own page with days since contact, their goal, next due date, and quick check-in buttons
- **Timeline** — every interaction gets logged with a timestamp, and you can filter by type, sort newest/oldest, or search by name/type/date

## Running it locally

```bash
npm install
npm run dev
```

Then go to localhost:3000
